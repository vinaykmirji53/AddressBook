5. Description:The Address Book project is a console-based application written in C language.
   It maintains and manages personal contact information including name, phone number, and email ID.

-> The program allows users to:
    *  Add new contacts with validations for name, phone number, and email ID.
    *  Search existing contacts using name, phone number, or email ID.
    *  Edit contact details based on search results.
    *  Delete contacts with confirmation (supports multiple matches).
    *  Display all contacts with sorting options (by name, phone, email, or normal order).
    *  This project strengthens understanding of strings, arrays, structures, and file handling in C.
